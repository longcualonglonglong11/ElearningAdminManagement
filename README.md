To run project:

At Frontend:
+ Go to frontend folder
+ Open terminal
+ cd/elearning && yarn && yarn start

At Backend:
+ Open project ElearningProject
+ Install Spring Dev 3 (If not install yet)
+ Run by Spring Boot
